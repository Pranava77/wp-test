# WEBFIXUS — WordPress theme

Neo-brutalist freelance-illustrator portfolio, built from the **WEBFIXUS Site** Claude Design as a
self-contained, Elementor-ready WordPress theme.

## Repo layout
- **`webfixus-build/webfixus/`** — the theme source (install this folder as a theme).
- **`webfixus-theme.zip`** — packaged theme, ready to upload via **Appearance → Themes → Add New → Upload**.

## Quick start
1. Install & activate the free **Elementor** plugin.
2. Upload `webfixus-theme.zip` → **Activate**.
3. On activation it auto-creates **Home / Work / About / Contact** as editable Elementor pages, sets Home as
   the front page, and builds the nav menu.
4. **Settings → Permalinks → Post name → Save**.

Full details, editing notes, and a QA checklist: [`webfixus-build/webfixus/README.md`](webfixus-build/webfixus/README.md).

Pages use Elementor's classic **Section → Column → Widget** structure (works whether or not the Flexbox
Container feature is enabled). Styling lives in `assets/css/webfixus.css`; interactions in
`assets/js/webfixus.js`.
